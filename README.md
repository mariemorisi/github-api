## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
```sh
git clone git@github.com:mariemorisi/github-api.git
```

2. Install all the dependencies with NPM or Yarn and run the application 

> Yarn :
```sh
yarn 
yarn start 
```
> NPM : 
```sh
npm i
npm start
```
